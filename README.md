# Usage
<!-- usage -->
```sh-session
$ npm install -g freemarker-visualizer
$ freemarker-visualizer COMMAND
running command...
$ freemarker-visualizer (-v|--version|version)
freemarker-visualizer/0.0.1 darwin-x64 node-v12.13.0
$ freemarker-visualizer --help [COMMAND]
USAGE
  $ freemarker-visualizer COMMAND
...
```
